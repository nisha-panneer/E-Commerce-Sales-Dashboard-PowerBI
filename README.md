# E-Commerce Sales Dashboard - Power BI

## Project Overview

The E-Commerce Sales Dashboard is an interactive Business Intelligence solution developed using Microsoft Power BI. This dashboard transforms raw e-commerce sales data into meaningful insights by analyzing revenue, profit, customer behavior, product performance, and sales trends.

The project demonstrates how data visualization can support strategic decision-making through interactive reports and key performance indicators (KPIs).
